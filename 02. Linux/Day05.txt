=> File permissions in Linux
-rw-r--r--. 1 ec2-user ec2-user 0 Apr  2 16:06 kastro.txt
drwxr-xr-x. 2 ec2-user ec2-user 6 Apr  2 16:07 day4

The file permissions are classified into 3 levels;
user level (u)
group level (g)
others level (o)

For each level, we will have 3 inbuild permissions;
read (r)
write (w)
execute (x)

rw-		r--		r--
u		g		o

chmod
chmod g+x kastro.txt
chmod o+wx kastro.txt

chmod u-r kastro.txt
chmod u-wx kastro.txt

Permissions with numbers;
0 - no permissions
1 - execute
2 - write
3 - 2+1 - write+execute
4 - read
5 - 4+1 - read+execute
6 - 4+2 - read+write
7 - 6+1 - read+write+execute
