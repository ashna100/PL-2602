Types of users in Linux;
By default when we launch an EC2 instance, we will get 2 types of users;
1. Normal user	- ec2-user (Amazon Linux), ubuntu (Ubuntu)
2. Root user		- root

Normal user has least privileges
root user has all kinds of privileges
